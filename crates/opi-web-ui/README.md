# opi-web-ui

Reusable web UI components for AI chat interfaces.

Part of the [opi](https://github.com/OdradekAI/opi) workspace — a Rust reimplementation of [pi](https://github.com/earendil-works/pi).

Provides web components for building AI-powered chat UIs with document preview support and multi-provider LLM integration. Built on [`opi-ai`](https://crates.io/crates/opi-ai).

## Status

Scaffolding release. The module structure (`components`) is in place; implementations are stubs.
