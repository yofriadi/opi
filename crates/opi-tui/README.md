# opi-tui

Terminal User Interface library with differential rendering.

Part of the [opi](https://github.com/OdradekAI/opi) workspace — a Rust reimplementation of [pi](https://github.com/earendil-works/pi).

Provides efficient text-based UI rendering that only updates changed regions, along with text editor components and markdown rendering.

## Status

Scaffolding release. The module structure (`render`, `editor`, `markdown`) is in place; implementations are stubs.
