# opi

AI agent toolkit in Rust — a reimplementation of [pi](https://github.com/earendil-works/pi). Source: [OdradekAI/opi](https://github.com/OdradekAI/opi).

## Crates

| Crate | Description |
|-------|-------------|
| `opi-ai` | Unified multi-provider LLM API with streaming support |
| `opi-agent` | General-purpose agent runtime with tool calling and transport abstraction |
| `opi-coding-agent` | Interactive coding agent CLI (`opi` binary) |
| `opi-tui` | Terminal UI library with differential rendering |
| `opi-web-ui` | Reusable web UI components for AI chat interfaces |

## Building

```sh
cargo build
```
